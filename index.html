Box Model

The box model is essentially a box that wraps around every HTML element. It consists of: borders, padding, margins, and the actual content.

Different parts of box model:

  • Content- the content of the box
  • Padding - clears an area around the content.It determines how elements look and sit within a container. It is transparent
  • Border - a border goes around the padding and content
  • Margin - clears an area outside the border. Margins are used to move an element up or down on a page as well as left or right. It is also transparent
